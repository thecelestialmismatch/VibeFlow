## Why VibeFlow?

VibeFlow is not just another compliance dashboard. It is an agentic framework capable of understanding regulatory documents (GDPR, HIPAA, SOC2), ingesting site specifics, identifying compliance gaps, and dynamically generating the exact legal policies and remedial web adjustments required. 

Instead of hiring expensive external compliance officers for your digital projects, VibeFlow offers an **"AI Compliance Officer for Businesses"** that does the heavy lifting via multi-agent RAG pipelines.

---

## How It Works

### 1. Initialize Scan
Enter any target URL and define the regulatory framework. VibeFlow's **Scout Agent** acts as a headless browser to gather all relevant context from your project.

### 2. Multi-Agent Analysis
The extracted content is routed into the **Gap Analyzer Agent** which cross-references standard regulations pulled from local vector embeddings or predefined logic (e.g., CCPA, EU AI Act).

### 3. Policy Generation & Remediation
The **Policy Generator Agent** constructs missing structural documents (Privacy Policies, Terms of Service) while the **Remediation Agent** highlights exact UI/UX code block adjustments needed to pass the audit.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth (Email / Password + OAuth)
- **Payments**: Stripe Checkout
- **Hosting**: Vercel
- **Styling**: Tailwind CSS + Shadcn UI
- **AI Core**: VoltAgent Framework

## Getting Started

1. Clone the repository.
2. Ensure you have the `.env.local` configured with your Supabase and Stripe keys.
3. Run `npm install` followed by `npm run dev`.
4. Open the application at `http://localhost:3000`.
